443 timeout

unable to access ‘https://github.com/Adam-hohai/leetcode.git/': Failed to connect to github.com port 443: Timed out
应该是dns解析错误，找最新的github的ip，因为有事会换
https://ipaddress.com/website/github.com

port出错

Failed to connect to 127.0.0.1 port 1080: Connection refused
昨天用ubuntu时，为了防止端口冲突把我windows的proxy端口换了没改回去(直接改clash里面的)
git config –global http.proxy http://127.0.0.1:当前proxy端口号
git config –global https.proxy http://127.0.0.1:当前proxy端口号
